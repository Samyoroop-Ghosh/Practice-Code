PS: ALL SELECTORS HAVE BEEN APPLIED TO THE P TAG

Descendant Selector (div p)

  • Selects all descendant elements under its parent

  • Even sub elements (ie. list elements will also be selected cuz they are descendants of div)

Child Selector (div > p)

  • Selects only direct child elements under it's parent.

  • Sub elements are ignored (ie. list elements inside a div will be ignored)

Sibling Selector (div ~ or + p)

  • Selects an element that is immediately following the specific element.

  • It matches the second element only when the element immediately follows thefirstelement, and both of them are the children of the same parent

  • We use ~ to select all the signs following the sibling and + to only select the immediate next occurence
